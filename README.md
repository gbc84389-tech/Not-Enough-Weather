# 🌤️ Not-Enough-Weather

Aplicação web de clima que consome APIs externas para exibir informações meteorológicas em tempo real com base na cidade informada pelo usuário.

---

## 🚀 Funcionalidades

- Busca de cidade via input
- Conversão automática de cidade → coordenadas (latitude/longitude)
- Exibição de:
  - Temperatura máxima
  - Sensação térmica
  - Direção do vento (convertida para cardinal)
  - Velocidade do vento
  - Precipitação
  - Probabilidade de chuva
- Data atual dinâmica
- Interface moderna com efeito glassmorphism

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- CSS Grid & Flexbox

---

## 🌐 APIs utilizadas

- **Geocoding API (Open-Meteo)**  
  Converte o nome da cidade em latitude e longitude

- **Meteoblue API**  
  Fornece os dados meteorológicos
